# CO-STAR Template — 內容創作

> 寫 email、blog、文案、社群貼文用。受眾與口吻是關鍵時必選。

```markdown
<context>
<!-- 背景：誰、做什麼產品、現在處於什麼階段 -->
</context>

<objective>
<!-- 想達成什麼具體成果 -->
</objective>

<style>
<!-- 寫作風格參考：「像 Stripe 公告」「像 Linear changelog」「像紐約時報專欄」 -->
</style>

<tone>
<!-- 情感色調：自信 / 謙遜 / 幽默 / 嚴肅 / 溫暖 / 急迫 -->
</tone>

<audience>
<!-- 對象是誰：年齡、職業、知識水平、與你的關係 -->
</audience>

<response>
<!-- 輸出格式：長度、段落數、是否含 CTA、HTML / Markdown / 純文字 -->
</response>
```

## 填寫提示

- **Style 與 Tone 差異**：Style 是「結構與用詞」，Tone 是「情感色調」。同一風格可以多種 tone。
- **Audience** 越具體越好：「30–45 歲台灣中小企業老闆，懂基礎財報但不懂 SaaS 訂閱模式」勝過「商務人士」。
- **Response** 最常被忽略：明寫長度、CTA、是否能用 emoji，避免 AI 亂發揮。
