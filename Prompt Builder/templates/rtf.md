# RTF Template — 簡單任務

> 一句話搞定、不需要繁複結構時用。最低 overhead。

```markdown
<role>
<!-- 一句話指派角色：regex 專家 / SQL DBA / 翻譯 -->
</role>

<task>
<!-- 一句話任務 -->
</task>

<format>
<!-- 一句話格式：JS / Markdown table / JSON / 純文字 -->
</format>
```

## 範例

```markdown
<role>regex 專家</role>
<task>寫一個 regex 比對台灣手機號碼（09 開頭 10 碼）</task>
<format>JS 寫法 + 一句話解釋</format>
```

## 填寫提示

- 最適合：翻譯、格式轉換、查詢小知識、單一 function 生成。
- 任務開始膨脹（要多步驟、要對風格、要 example）→ **換 RISEN 或 TIDD-EC**。
- Role 沒想法 → 留空也行；多數情況 task + format 兩欄就夠。
